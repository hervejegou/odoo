Sign in with Auth0

The Auth0 OAuth2 Login for Odoo module enables seamless and secure user authentication in Odoo using Auth0 credentials. By integrating with Auth0’s OAuth2 protocol, this module allows users to log in through a trusted third-party identity provider, removing the need for local passwords and enhancing security. Upon login, users are redirected to Auth0’s secure interface, and after successful authentication, they are either logged into an existing Odoo account or automatically registered using profile details like name and email. This integration is ideal for businesses and platforms that use Auth0 for centralized identity management, offering a streamlined Single Sign-On (SSO) experience and supporting token-based authentication flows within Odoo.

Generate Credentials in Auth0

Go to https://auth0.com/. Sign up for a new account if you don't have one, or log in if you already do.

Once logged in, navigate to the Dashboard.


In the left-hand sidebar, click on Applications, then click on Applications again from the dropdown.

Create an Application in Auth0

Click the Create Application button.

Choose the application type (e.g., Regular Web Applications) and provide a name for your application.


Click on Create to proceed.

Configure the Application

After creating your application, go to the Settings tab.

In the Application URIs section, enter the Allowed Callback URLs. This is the URL to which Auth0 will redirect after successful authentication.

For example: https://your-odoo-instance/auth_oauth/signin


Enter the Allowed Logout URLs.


For example: https://your-odoo-instance/web/logout


You can also add Allowed Web Origins if needed (e.g., for frontend integrations).

Credentials

In the Basic Information section, you will find the Client ID, Client Secret, and Domain. Copy these values — they are required to integrate Auth0 with Odoo.

Configuring OAuth Providers

Navigate to settings and under the Developer Tools, Click on Activate the developer mode.

Navigate to settings > Users & Companies > Click on OAuth Providers

Click on the Sign in with Auth0

Generate and enter your Auth0 credentials (Client ID, Secret Key, Domain) and mark allowed true, Save the changes in Record.

Creating New user with Auth0

Click on Log in with Auth0

Upon successful authentication, New User will be created for the first time login and later users can log in with Auth0.

Key Features

Enables Auth0 OAuth2 login for Odoo with secure JWT token decoding.

Automatically creates user accounts in Odoo from Auth0 profile data.

Eliminates the need for passwords using secure OAuth-based login.

Quick and easy to configure Auth0 credentials from Odoo backend.

Secure login and retrieval of user details like name and email from Auth0.
